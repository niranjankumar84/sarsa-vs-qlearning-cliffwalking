# SARSA vs Q-Learning Cliff Walking Agent 🤖

This project demonstrates the implementation of two classic Reinforcement Learning algorithms:

- SARSA
- Q-Learning

using the Cliff Walking environment from Gymnasium.

The agent learns through trial and error by interacting with the environment, receiving rewards/penalties, and updating Q-values over multiple episodes.

---

## 📌 Features

- SARSA implementation from scratch
- Q-Learning implementation from scratch
- Q-table based learning
- Epsilon-Greedy exploration strategy
- Cliff Walking environment visualization
- Reward tracking
- Policy learning through episodes

---

## 🧠 Concepts Covered

- Reinforcement Learning
- Q-Table
- Temporal Difference Learning
- Exploration vs Exploitation
- Bellman Equation
- State-Action Value Function
- Safe vs Optimal Policies
